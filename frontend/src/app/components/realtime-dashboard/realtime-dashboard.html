1Angular connects to /hubs/dashboard with JWT token
2OnConnectedAsync() fires — server notifies all clients
3Background service broadcasts stats every 2s to all clients
4Angular hubConnection.on('StatsUpdated') receives data
5UI updates automatically — no polling, no page refresh
6OnDisconnectedAsync() fires on tab close/logout
